Operating Systems in Java

A collection of Java implementations of classic Operating Systems concepts and algorithms.

This repository serves as a learning project where I implement and experiment with synchronization, scheduling, memory management, and other fundamental Operating Systems topics while studying the subject.

The objective is to better understand how these concepts work in practice by translating textbook algorithms into executable Java programs.

Purpose

This repository is intended as a practical companion to Operating Systems coursework, documenting implementations and experiments as I continue learning the subject.
